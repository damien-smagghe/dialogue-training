# Todo List App

A simple React todo list application built with Vite.

## Deployed App

[Visit the App](https://damien-smagghe.github.io/dialogue-training/)

## Features

- Add new todo items
- Mark todos as complete/incomplete
- Delete todo items
- Clean, modern UI

## Local Development

```bash
npm install
npm run dev
```

## Building for Production

```bash
npm run build
```

The build folder will be automatically deployed to GitHub Pages.
